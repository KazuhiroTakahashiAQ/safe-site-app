# 事故事例提案くん

建設現場の工程に応じて、当日起こり得る災害事例を確認できるデモアプリです。

## GitHub Pages

- https://KazuhiroTakahashiAQ.github.io/safe-site-app

## 主な機能

- 工程表（ガントチャート）上で日付ラインを動かして事例を抽出
- 選択日の事例一覧（件数・画像枚数付き）
- 画像クリックで拡大表示、モーダル内でページ送り
- 災害事例ライブラリ

## ローカル開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## GitHub Pages デプロイ

```bash
npm run deploy
```
